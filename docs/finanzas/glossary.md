# Finanzas SD – Glossary / Glosario bilingüe

**Last updated:** 2025-12-06  
**Audience:** All users (PMO, Finanzas, SDMT, Audit)  
**Purpose:** Bilingual terminology reference

| Término | English | Descripción |
| --- | --- | --- |
| Baseline | Baseline | Versión aprobada de presupuesto inicial de un proyecto. |
| Rubro | Cost category | Entrada del catálogo SDMT con código, tipo de costo y unidad. |
| Line item | Line item | Partida derivada de un rubro dentro de un proyecto. |
| Handoff | Handoff | Entrega formal del estado del proyecto a SDMT. |
| Evidence | Evidencia | Documento o captura que respalda una factura o línea. |
| Allocation rule | Regla de asignación | Parámetros para distribuir montos por periodo. |
| Close month | Cierre mensual | Proceso que bloquea edición y consolida variaciones. |
| Reconciliation | Conciliación | Comparación de forecast contra facturas reales. |
| Alert | Alerta | Notificación de desviación o pendiente de aprobación. |
| PMO | Project Management Office | Oficina de proyectos responsable de intake y baseline. |
| SDMT | Service Delivery Management Team | Equipo que consume el handoff y controla ejecución. |
| FIN | Finance | Área que maneja forecast, facturas y cierres. |
| AUDIT | Audit | Equipo que revisa evidencias y cambios. |
