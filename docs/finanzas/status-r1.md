# Finanzas SD – Estado R1

**Last updated:** 2025-12-06  
**Audience:** PMO, SDMT, Executives  
**Purpose:** R1 release status and scope summary  
**Note:** This document reflects R1 initial release scope. Current features exceed this baseline.

## Completado (R1 Core)
- **Proyectos (Projects)**: creación/listado y handoff a SDMT.
- **Catálogo de Rubros (Rubros Catalog)**: consulta de catálogo maestro con taxonomía.
- **SDMT Catalog**: sincronizado para selección y forecast.
- **Conciliación (Reconciliation)**: registro de facturas/prefacturas y asociación a line items.
- **Carga de documentos**: soporte vía `/uploads/docs` (module=`reconciliation`).
- **Reglas de Asignación (Allocation Rules)**: definición de reglas de distribución.
- **Ajustes Presupuestarios (Adjustments)**: gestión de excesos, reducciones y reasignaciones.
- **Flujo de Caja (Cashflow)**: monitoreo de ingresos, egresos y margen mensual.
- **Escenarios (Scenarios)**: comparación de escenarios frente al baseline.
- **Proveedores (Providers)**: registro y administración de proveedores.

## En progreso (Post-R1)
- Integración de baseline de prefactura para PMO (alinear montos iniciales).
- Reportes adicionales de avance y controles ejecutivos.
- Mejoras en visualización de cambios y alertas.

## Diferido / Futuro
- Automatización de alertas operativas extendidas.
- Mejoras de forecast avanzado y cierres mensuales automáticos.
- Paneles adicionales para auditoría detallada.
- Integración con sistemas externos de facturación.
