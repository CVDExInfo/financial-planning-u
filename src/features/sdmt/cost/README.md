# SDMT Cost – Finanzas SD context

- Esta carpeta contiene módulos de UI que exponen el catálogo SDMT y la conciliación utilizada por Finanzas SD.
- Catalog y Reconciliation se alimentan de Finanzas API (`services/finanzas-api`) para mostrar rubros, line items y facturas.
- Use esta UI para seleccionar rubros del catálogo SDMT, revisar forecast y asociar facturas/soporte antes de handoff.
- Documentación funcional y técnica: consulte `docs/finanzas/overview.md` y `docs/finanzas/architecture.md`.
